
# habiFlow-desktop

- React 18 + Vite + TypeScript 기반 데스크탑 앱 초기 템플릿
- Zustand 기반 상태관리, React Query, React Hook Form 포함
- TailwindCSS + shadcn/ui 스타일링 적용
- 간단한 로그인 UI 및 상태관리 예제 포함

## 시작하기

```bash
pnpm install
pnpm dev
```

## 주요 라이브러리

- react, react-dom
- vite
- typescript
- zustand
- @tanstack/react-query
- react-hook-form
- tailwindcss
- shadcn/ui (커스텀 Button 예제 포함)
